Geocoding API version

Go to SearchBox version

Pro:

Free version supports up to 100K request.
Can search using long, lat info (eg: searching -122.994560, 49.246445 will bring to Burnaby)

Con:

Less comprehensive search. Can find locations based addresses, cities. But is not that great at finding location based on name of place (eg: if you search BCIT, you will not find the correct location)
